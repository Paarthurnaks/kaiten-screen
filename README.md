# Kaiten Screen

Windows-приложение для быстрого захвата скриншотов и создания по ним задач в
[Kaiten](https://kaiten.ru/). Живёт в трее, ничего лишнего на экране не показывает,
пока не понадобится.

Сценарий использования: увидели баг или что-то, что нужно обсудить с командой →
нажали хоткей или иконку в трее → выделили область экрана → выбрали, что сделать со
скриншотом (создать новую задачу в Kaiten, прикрепить к уже существующей или просто
скопировать в буфер обмена).

## Возможности

- **Захват области экрана** — выделение мышью с изменяемыми размерами (ручки по
  углам) и перетаскиванием готовой области целиком (как в Lightshot).
- **Создание задачи в Kaiten** — форма с выбором пространства, доски, дорожки,
  колонки, ответственного, участников и пользовательских полей; скриншот
  автоматически прикрепляется к созданной карточке.
- **Прикрепление к существующей задаче** — поиск карточки по названию/ID и
  прикрепление скриншота к ней.
- **Просто скопировать в буфер обмена** — если Kaiten не нужен прямо сейчас,
  приложением можно пользоваться как обычным скриншотером.
- **Глобальный хоткей и трей** — левый клик по иконке в трее сразу открывает
  выделение области, правый клик — меню (скриншот / настройки / проверить
  обновления / выход).
- **Автообновление** — приложение само проверяет новые версии на GitHub Releases,
  скачивает их в фоне и устанавливает при следующем запуске.
- **Автозапуск с Windows** — включается в настройках.

## Хоткей по умолчанию

`Ctrl+Shift+K` — запускает выделение области экрана из любого места (не нужно
переключаться на приложение). Изменить можно в Настройках → раздел «Захват»;
изменение применяется сразу, без перезапуска.

Во время выделения области:
- `Esc` — отменить захват.

## Настройки

Открываются через трей → «Настройки». Три раздела:

- **Подключение к Kaiten** — домен вашего Kaiten (например `mycompany.kaiten.ru`) и
  API-ключ. Кнопка «Проверить соединение» подгружает список пространств/досок —
  без неё выпадающие списки в разделе «Значения по умолчанию» останутся пустыми.
- **Значения по умолчанию** — пространство, доска, колонка, дорожка и ответственный,
  которые будут подставляться в форму создания задачи, чтобы не выбирать их каждый
  раз заново.
- **Захват** — хоткей запуска и автозапуск приложения вместе с Windows.
- **Файл настроек** — экспорт и импорт настроек одним файлом (см. ниже).

### Экспорт / импорт настроек

В настройках можно сохранить текущую конфигурацию (домен, значения по умолчанию,
API-ключ) в JSON-файл через обычный диалог «Сохранить как…» — удобно, чтобы не
вводить всё заново на новом компьютере или поделиться готовой настройкой с
коллегой. Импорт делается через диалог «Открыть файл…» и сразу применяет
настройки поверх текущих.

⚠️ Экспортированный файл содержит API-ключ в открытом виде — храните и передавайте
его так же осторожно, как сам ключ.

## Установка и обновление

Установщик под Windows (`.exe`) доступен на странице
[Releases](../../releases) этого репозитория. Приложение проверяет обновления
автоматически при запуске и раз в сессию через трей → «Проверить обновления»;
скачанное обновление ставится при следующем запуске приложения.

## Разработка

Стек: Electron + React + TypeScript, гексагональная архитектура
(domain → application → infrastructure/main → renderer).

```
npm install
npm run dev          # запуск в режиме разработки
npm run typecheck
npm run lint
npm run test          # юнит-тесты (vitest)
npm run test:e2e       # e2e-тесты (playwright, мок-сервер Kaiten API)
npm run package        # собрать установщик локально, без публикации
```
